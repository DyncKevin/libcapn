Libcapn Documentation

libcapn is a C Library to interact with the Apple Push Notification Service using simple and intuitive API.With the library you can easily send push notifications to iOS and OS X (>= 10.8) devices.

The library is licensed under the MIT license.

Note
libcapn uses Jansson library code to create JSON document.

Grab the source code

The source repository is hosted at GitHub.

You can retrieve the source code through the standard Git client with:

git clone git://github.com/adobkin/libcapn libcapn
cd libcapn
git submodule update --init

You can also download release tarballs from FTP and HTTP.
